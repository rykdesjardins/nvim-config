# Modern Neovim Config

A highly opinionated, modern Neovim configuration built with **Lua**, **Lazy.nvim**, and **Snacks.nvim**.

This setup is optimized for **TypeScript/Web Development** (using Biome and standard TS server) and features a polished UI with the **Catppuccin** theme.

## Features

- **Package Manager**: [lazy.nvim](https://github.com/folke/lazy.nvim) for lightning-fast startup.
- **LSP & Formatting**: Auto-configured TypeScript & Biome support via [Mason](https://github.com/williamboman/mason.nvim).
- **UI & Navigation**: [Snacks.nvim](https://github.com/folke/snacks.nvim) for fuzzy finding, git integration, and refined UI elements.
- **Buffers**: Tab-like buffer management with [barbar.nvim](https://github.com/romgrk/barbar.nvim).
- **AI Integration**: [Opencode](https://github.com/NickvanDyke/opencode.nvim) built-in for in-editor assistance.
- **Eye Candy**: Neovide support with "Pixiedust" cursor effects and Comic Mono font configuration.

## Prerequisites

*   **Neovim**: v0.9.0 or higher (LuaJIT required).
*   **Font**: A Nerd Font is required for icons. The config is pre-set for [Comic Mono](https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/ComicShannsMono), but any Nerd Font works.
*   **Ripgrep**: Required for fast text searching (used by Snacks Picker).
*   **Node.js**: Required for Mason LSP servers.
*   **C Compiler**: (gcc/clang) for compiling Treesitter parsers.

## Installation

### 1. Backup existing config
If you have a current configuration, back it up first:
```bash
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
```

### 2. Clone the repository
Clone directly into your configuration directory. This allows you to update simply by running `git pull`.
```bash
git clone https://github.com/rykdesjardins/nvim-config ~/.config/nvim
```

### 3. Start Neovim
Run `nvim`. Lazy.nvim will automatically bootstrap and install all plugins.
```bash
nvim
```
*Note: The first startup might take a minute as plugins install. Restart Neovim once fully installed.*

## Keymaps

**Important:** The Leader key is mapped to **`#`** (Shift + 3).

### General
| Key | Action |
| :--- | :--- |
| `#` | **Leader Key** |
| `<C-n>` | Toggle File Explorer (NvimTree) |
| `<C-s>` | Save File |
| `<A-,>` | Previous Buffer |
| `<A-.>` | Next Buffer |
| `<A-c>` | Close Buffer |
| `<A-S-c>` | Restore Buffer |

### Fuzzy Finding (Snacks)
| Key | Action |
| :--- | :--- |
| `#ff` | Find Files |
| `#fg` | Grep (Search Text) |
| `#fw` | Grep Word under cursor |
| `#fr` | Recent Files |
| `#fb` | Open Buffers |
| `#gb` | Git Branches |

### LSP & Coding
| Key | Action |
| :--- | :--- |
| `gd` | Go to Definition |
| `gr` | Find References |
| `#ac` | Code Action |
| `#rn` | Rename Symbol |
| `K` | Hover Documentation |
| `<C-c>` | Accept Copilot Suggestion |

### AI Assistant (Opencode)
| Key | Action |
| :--- | :--- |
| `<C-a>` | **Ask** (Open prompt) |
| `<C-x>` | **Select** (Execute action) |
| `go` | Add range to context |

## Neovide Settings
If you use [Neovide](https://neovide.dev/), this config enables:
- **Font**: Comic Mono
- **Padding**: Clean internal padding
- **VFX**: "Pixiedust" particle effects on cursor movement

## Customization
This configuration is a single-file setup in `init.lua` for simplicity, though it loads plugins dynamically.
- **Theme**: Change line 479 (`vim.cmd.colorscheme("catppuccin-mocha")`) to swap themes.
- **LSP**: Add servers to the `mason-lspconfig` setup block on line 321.
